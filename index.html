Repository contents

  • data/generated_layers/*.json - one generated layer per included group
  • data/iran_attack_aggregated_layer.json - combined ATT&CK Navigator layer
  • data/manifest.json - complete machine-readable dataset for the site
  • data/enterprise-attack-v18.1-*.xlsx - MITRE ATT&CK source files used to build the repo
  • scripts/build_repo.py - reproducible regeneration script
