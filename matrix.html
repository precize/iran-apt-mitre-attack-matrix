How this page works

Each column is an Enterprise ATT&CK tactic. Each tile is a technique observed across the selected Iran-linked groups. The number in parentheses shows how many groups in this repository use that technique. Clicking a tile opens the official ATT&CK technique page.

Sub-techniques can be hidden to keep the matrix readable or shown to expand the full mapped set. The compact layout reduces tile height for denser browsing.
